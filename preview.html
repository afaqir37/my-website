On-demand car wash platform · Sole Engineer · 2023–2024

A two-sided marketplace connecting customers with car washers in Morocco. Users book washes at their location, washmen receive real-time requests based on proximity, payments happen in-app. I built everything — both mobile apps, the backend, the admin dashboard.

Authentication without friction

Users sign in with just their phone number — no passwords to remember, no SMS codes to wait for. The backend generates secure credentials using crypto.randomBytes and handles all AWS Cognito operations server-side.

Geolocation that doesn't kill batteries

The first version calculated distances on the client. It worked, but drained batteries and burned through API quota. I moved everything to PostGIS on the server — now Socket.io only pushes requests to washmen who are actually nearby. 3x faster, 80% fewer API calls.

Payments in Morocco

Integrated CMI, Morocco's payment gateway. The full flow: generating order IDs with nanoid, signing requests with HMAC-SHA512, handling WebView payments, verifying callbacks, preventing double-spending.

Notifications that don't miss

Washmen can receive multiple booking requests simultaneously. Built a queue system with deduplication, auto-rotation every 12 seconds, swipe-to-dismiss with gesture handlers, and haptic feedback. No booking gets lost.

Built with React Native, Expo, TypeScript, Node.js, PostgreSQL, PostGIS, Socket.io, AWS Cognito, S3
